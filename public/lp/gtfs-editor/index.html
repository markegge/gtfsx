FAQ

Is it really free?
Yes. The editor is free forever. You only pay if you want the Planner plan: hosted publishing, snapshot history, and the service-planning suite.
Do I need to sign up?
No. You can build, validate, and export a feed without an account. Sign up only if you want to sync across devices or use paid features.
What's the difference between the free Editor and Planner?
The Editor is free: unlimited editing, validation, and export to .zip. Planner, the service-planning suite for transit agencies, adds hosted publishing (your feed at a canonical URL), snapshot history, Mobility Database submission, and the planning analyses.
Can I import an existing feed?
Yes. Drag a .zip into the editor. We parse routes, stops, calendars, fares, and shapes.
Does GTFS-Flex really work?
Yes. Polygon zones, booking rules, stop groups, and the full Flex extension. See the GTFS-Flex guide for a walkthrough.
What if my feed has problems?
The validator catches structural and reference errors as you edit. For specific issues, see the validation docs.
